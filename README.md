# Digital Memory Capsule

Digital Memory Capsule is a secure web platform that allows users to store messages, photos, and videos inside a digital capsule that unlocks on a future date. The system is designed to preserve memories, emotions, and important moments securely over time.

## Features

- Secure user authentication
- Create time-locked memory capsules
- Upload photos, videos, and text messages
- Countdown timer for capsule unlock
- Planned AES-256 encryption support
- Email reminder notifications
- Share capsules with friends or family
- Responsive modern UI

## Tech Stack

### Frontend
- React
- Vite

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Authentication & Security
- JWT Authentication
- bcrypt Password Hashing
